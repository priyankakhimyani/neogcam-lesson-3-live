Open Source: A guide for beginners

What is open source

We have been using open source since we were a kid for eg: VLC, Google Chrome, Firefox etc.

Open-Source Projects are projects which are readily available, free to use and free to modify. Open-Source Contribution is basically contributing to the improvement and development of open-source software.

Some popular Open-Source Software are Android, Android, Linux Operating System, Chromium and VSCode IDE

Why contributing is important

Whether you a student or a working professional open source is a great way to get real world software development experience from the comfort of your home. It helps in strengthening the understanding of the project. You will be able to understand how teams work in an organization.

You get to help people within the community and create an impact. You also get recognized for your work, this recognition can also end up with a lot of opportunities by potential employers.

If you want to get a good job opportunity without learning DSA, open is a great way to get such opportunities by doing great work in public

Version Control Systems(VCS)

They are software tools which help us to store all the changes in order to keep a track of them. These changes in the source code are called different versions and we can also revert to an older version if required at point in time.

Some examples of VCS are Git, Mercurial, CVS etc.

Git is the most widely used VCS. GitHub is the place where you keep all your projects and multiple people can work together on those projects

How to get started

Open source your projects

Showcase your projects on GitHub, this can help you in getting contributions from the community. This will help you to collaborate with developers across the world. Not only this but it will also add value to your project

Start with beginner friendly issues

Go on GitHub and search easy or beginner-friendly issues. Here are some links which I feel will be helpful. First Timers Only issues Easy Issues

You can also find projects or organizations that you are interested in by going to their GitHub repository and searching for easy issues.

You will fill find links to their Discord or slack on Github. Introduce yourself and ask for help whenever you are stuck.
